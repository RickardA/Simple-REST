Consuming the simple REST-API

To perform simple CRUD operations follow the simple steps below

  • Create a movie

    • Request Type
      Post
    • URL Path
      /api/movies
    • Body
      object containing the following properties, no property is required
      {
      title: String,
      rating: float,
      description: String,
      imageUrl: String
      }
    • Returns
      Success:
      Status Code: 201, Body: The saved movie.
      Error:
      Status Code: 500, Body: Nothing.

  • Get all movies

    • Request Type
      Get
    • URL Path
      /api/movies
    • Returns
      Success:
      Status Code: 200, Body: Array of all movies.
      If no movies exists:
      Status Code: 204, Body: Nothing.
      Error:
      Status Code: 500, Body: Nothing.

  • Get movie by id

    • Request Type
      Get
    • URL Path
      /api/movies/[putMovieIdHere]
    • Returns
      Success:
      Status Code: 200, Body: Requested movie.
      If movie does not exist:
      Status Code: 204, Body: Nothing.
      Error:
      Status Code: 500, Body: Nothing.

  • Update a movie by id

    • Request Type
      Put
    • URL Path
      /api/movies
    • Body
      Get existing movie and use following object structure, change properties you want to change and make a put request
      {
      _id: String,
      title: String,
      rating: float,
      description: String,
      imageUrl: String
      }
    • Returns
      Success:
      Status Code: 200, Body: Updated movie
      If no movie exists:
      Status Code: 400, Body: Nothing.
      Error:
      Status Code: 500, Body: Nothing.

  • Delete movie by id

    • Request Type
      Delete
    • URL Path
      /api/movies/[putMovieIdHere]
    • Returns
      Success:
      Status Code: 200, Body: Nothing
      If movie does not exist:
      Status Code: 200, Body: Nothing.
      Error:
      Status Code: 500, Body: Nothing.
